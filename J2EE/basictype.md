# 基本数据类型
## 占用信息表

| 基本类型  | 大小(bit) | 最大值  | 最小值   | 包装器类型 |
| --------- | --------- | ------- | -------- | ---------- |
| `boolean` | -         | -       | -        | Boolean    |
| `char`    | 16        |         |          | Character  |
| `byte`    | 8         | -128    | +127     | Byte       |
| `short`   | 16        | -2^15^  | +2^15^-1 | Short      |
| `int`     | 32        | -2^31^  | +2^31^-1 | Integer    |
| `long`    | 64        | -2^63^  | +2^63^-1 | Long       |
| `float`   | 32        | IEEE754 | IEEE754  | Float      |
| `double`  | 64        | IEEE754 | IEEE754  | Double     |
| `void`    | -         | -       | -        | Void       |

- bit就是位，byte就是字节 ，1byte=8bit
