# VUE

[中文官网](https://cn.vuejs.org/)

## 简介

一套用于构建用户界面的**渐进式框架**。适合于单页面的后台前端创建。

## 安装

```bash
> npm install -g vue
```

## vue-cli

[中文官网](https://cli.vuejs.org/zh/)

- 安装

```bash
> npm install -g @vue/cli
> yarn global add @vue/cli
```

- 版本检测

```bash
> vue --version
```

- 创建项目

```bash
> vue create hello-world
```

- 图形化界面

```bash
> vue ui
```

## 目录结构

```bash
project
	|
	|-- dist ## build生成打包文件
	|—— node_modules ## 依赖包
	|-- public ## 静态页面文件
	|-- src ## 源码
	|-- babel.config.js
	|-- package.json
	|-- package-lock.json
	|-- README.md
	|-- vue.config.js
    └─
```
