# HTML

## 简介

## 标签

```html
<html>
<body >

<h1 >我的第一个标题</h1>

<p>我的第一个段落。</p>
<div class="类标签属性，用于css样式统一以及js" id="用于标识单个元素" style="单个元素样式覆盖父类样式等"></div>
</body>
</html>
```

```html
<h1>标题,h1最大，h5最小，上下的间隔较大</h1>
<p>段落</p>
<hr /> <!-- 分隔线 -->
<pre>格式化文本段落</pre>
<div>块容器</div>
<a href="链接地址">标签(标题)</a>
<img src="图片.jpg" width="104" height="142" />
<br /> <!-- 手动换行 -->
<!-- 表格 -->
<table>
    <
</table>
```

- 文本格式化标签

  | 标签     | 描述           |
  | :------- | :------------- |
  | <b>      | 定义粗体文本。 |
  | <big>    | 定义大号字。   |
  | <em>     | 定义着重文字。 |
  | <i>      | 定义斜体字。   |
  | <small>  | 定义小号字。   |
  | <strong> | 定义加重语气。 |
  | <sub>    | 定义下标字。   |
  | <sup>    | 定义上标字。   |
  | <ins>    | 定义插入字。   |
  | <del>    | 定义删除字。   |
